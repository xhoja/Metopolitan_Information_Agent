## Log Format

Each member logs findings in their personal wiki page under a `## Research Log` section. Entries should include:

- Date
- What was built / tested
- What worked, what didn't
- Key finding or recommendation (1–2 sentences)
- Link to any proof-of-concept code (branch or folder in repo)
